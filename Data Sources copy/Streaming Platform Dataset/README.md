# Streaming Platform Comprehensive Dataset

## Dataset Overview
This dataset combines catalog information from major streaming platforms with financial metrics, content spending data, and industry projections. Created: 2026-02-15

## Data Sources

### Financial Data Sources (Q4 2025 - Q1 2026)
- Netflix Q4 2025 Earnings Report [citation:5]
- Roku Q4 2025 Earnings Report [citation:1][citation:10]
- AMC Networks Q4 2025 Earnings [citation:2]
- Ampere Analysis Global Content Spending Report [citation:7]
- Ampere Analysis Sports Rights Report [citation:3]
- Digital TV Research Subscriber Forecasts [citation:9]
- ITVX Streaming Data [citation:4]
- Twitch Viewership Data [citation:8]

## Files Included

1. **streaming_catalog_enriched.csv** - Main dataset with catalog + financials
2. **platform_financials_comprehensive.csv** - Detailed platform financial metrics
3. **platform_summary.csv** - Platform overview with key characteristics
4. **industry_metrics.csv** - Industry-wide metrics and projections
5. **title_financial_estimates.csv** - Title-level estimates from analyst reports

## Key Metrics Covered

- Subscriber counts (actual and projected)
- Quarterly revenue and profit
- Content spending (annual and by platform)
- Sports rights spending
- Advertising revenue
- Streaming hours and viewership
- Platform launch dates and parent companies
- Title-level estimates where available

## Important Notes

- Financial data comes from Q4 2025/Q1 2026 earnings reports
- Subscriber projections are analyst estimates [citation:9]
- Title-level financials are estimates from analysts, not official disclosures
- Some platforms have limited public financial data (Apple TV+, Peacock)
- Sports rights data from Ampere Analysis [citation:3]
- Catalog data may not include very recent additions

## Usage Examples

- Analyze content spending vs. subscriber growth
- Compare platform financial health
- Identify content investment trends
- Calculate estimated ROI for specific titles (where data available)
- Study the relationship between catalog size and financial performance
